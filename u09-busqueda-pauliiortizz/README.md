# Unidad 9 - Algoritmos de Busqueda

## Ejercicio N°1

valor en un arreglo

## Ejercicio N°2

## Ejercicio N°3

teclado vaya generando un árbol balanceado. Modificar la Clase Árbol vista en clase.
